◆ ROUND5_API_TAU2_LEN

#define ROUND5_API_TAU2_LEN   (1<<11)

Defines the length of the random vector for the A matrix when A is created using TAU=2.

The value 0 (the default) means that the value of parameter 2^11 will be used.

Important: this must be a power of 2 and larger than parameter d!

Adds the value of the message to first len coefficients of a matrix.

The message is interpreted as a bit string where each group of bits_coeff bits is first scaled by the scaling factor and then added to the coefficient.

Parameters
[out]resultresult of the addition
[in]lenlength of the result
[in]matrixmatrix to which add the message
[in]mmessage to add
[in]bits_coeffnumber of bits added in each coefficient
[in]scaling_factorscaling factor applied (also defines the modulo as 2^scaling_factor)
Returns
0 in case of success
